## рҹӣё What is This?

**LeaderBoard Frontend** is the visual powerhouse behind **BugByte 2026** вҖ” Ignite Club's flagship competitive programming contest. It's not just a leaderboard. It's an *experience*.

Built with a **deep space aesthetic** вҖ” animated starfields, matrix rain, holographic tilt cards, podium effects, and slot machine rank animations вҖ” this UI makes every rank feel earned and every update feel cinematic.

> Ranked. Animated. Deployed. Out of this world.

## рҹҡҖ Getting Started

### Prerequisites

```bash
node --version   # v18+ required
npm --version    # v9+ recommended
```

### Installation & Dev

```bash
# 1. Clone the repository
git clone https://github.com/aarav12e/LeaderBoard_Frontend.git
cd LeaderBoard_Frontend

# 2. Install dependencies
npm install

# 3. Set up environment variables
cp .env.example .env
# вҶ’ Fill in VITE_API_BASE_URL

# 4. Fire up the dev server
npm run dev
```

The app launches at `http://localhost:5173` рҹҡҖ

### Build for Production

```bash
npm run build       # Outputs to /dist
npm run preview     # Preview the production build locally
```

## вҳҒпёҸ Deployment

Auto-deployed to **Vercel** on every push to `main`.

```
рҹ“Ұ git push origin main
        в”Ӯ
        в–ј
рҹ”Ғ Vercel detects changes
        в”Ӯ
        в–ј
вҡЎ vite build
        в”Ӯ
        в–ј
рҹҢҗ Live at leader-board-frontend-two.vercel.app
```

The `vercel.json` handles SPA routing вҖ” all paths redirect to `index.html` so React Router works properly on refresh.

```json
{
  "rewrites": [{ "source": "/(.*)", "destination": "/index.html" }]
}
```

### Deploy your own fork

```bash
npm i -g vercel
vercel login
vercel --prod
```

> Set `VITE_API_BASE_URL` in **Vercel Dashboard вҶ’ Settings вҶ’ Environment Variables**.
